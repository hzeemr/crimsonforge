// CrimsonForge Quest Dialog Tracer Script for x64dbg
//
// This script traces the HasQuestDialog gimmick state check
// to find exactly where the character ownership comparison happens.
//
// Usage:
//   1. Launch Crimson Desert, load save as Damiane
//   2. Go near a quest NPC with ! marker (switch to Kliff to verify it exists, then switch back to Damiane)
//   3. Open x64dbg -> File -> Attach -> select CrimsonDesert.exe
//   4. Go to Script tab -> Load script -> select this file
//   5. Run the script
//   6. Switch back to the game
//   7. Walk near the quest NPC
//   8. Switch back to x64dbg to see the log

// Set logging
log "=== CrimsonForge Quest Dialog Trace ==="
log "Looking for HasQuestDialog evaluation..."

// The gimmick state machine state names are in the exe
// "HasQuestDialog" string is at RVA 0x4AB6AD8 (approx)
