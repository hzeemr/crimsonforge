CrimsonForge Character Unlock v10
==================================

Unlocks all playable characters for all chapters in Crimson Desert.

WHAT IT DOES:
- Removes "This is Kliff's story" popup (hidden via HTML patch)
- Removes "Comrade is on an important mission" lock (runtime vtable patch)
- Removes movement freeze when spawning locked characters (stageinfo patch)
- Removes 29+ condition locks (conditioninfo patch)
- Removes 185 NPC interaction locks (gimmickgroupinfo patch)
- Damiane available from Chapter 3 onward (all chapters)
- Oongka available from Chapter 7 onward (all chapters)

TWO COMPONENTS:

1. DATA PATCHES (permanent, applied via CrimsonForge):
   - stageinfo.pabgb: 14 Forbidden→Permitted entries
   - conditioninfo.pabgb: 29 CheckCharacterKey stripped
   - gimmickgroupinfo.pabgb: 185 CheckCharacterKey stripped
   - changecharacternoticepanel.html: popup hidden + no game pause

   Apply: CrimsonForge → Explorer → Quick Mods → Remove Condition Locks
   These survive game restarts. Only wiped by Steam "Verify Integrity".

2. RUNTIME PATCH (per-launch via x64dbg OR ASI):
   The forbidden character list is generated by game code.

   Option A — x64dbg (no ASI loader needed):
   1. Launch game to MAIN MENU (don't load save yet)
   2. Attach x64dbg (with ScyllaHide, Denuvo x64 profile)
   3. Command: disasm [CrimsonDesert.exe+4843218]
   4. Space → xor rax, rax → OK → ret → OK
   5. F9 → load your save
   6. All characters available

   Option B — ASI Plugin (via CDUMM):
   1. Install CDUMM (nexusmods.com/crimsondesert/mods/84)
   2. CDUMM → ASI Plugins → Install → CrimsonForgeCharUnlock.asi
   3. Launch game — patches applied automatically

OONGKA NOTE:
Oongka requires completing faction quest "Words Left by the Riverside".
Use CrimsonSaveEditor (github.com/NattKh/CRIMSON-DESERT-SAVE-EDITOR)
Quest Editor to mark it complete, or do the 10-min fishing quest in-game.

GAME UPDATES:
- ASI checks RTTI before patching — safely disables on version mismatch
- Data patches may need re-applying after Steam verification
- Check bin64/crimson_charunlock.log for ASI status

UNINSTALL:
- Steam → Properties → Verify Integrity (restores all files)
- Delete ASI from CDUMM or bin64/scripts/

Save file location: %LOCALAPPDATA%\Pearl Abyss\CD\save\
Save format: ChaCha20 encrypted + LZ4 compressed (use CrimsonSaveEditor)
